% Abstract English:
% Structure per Chi review: problem - solution - result. No related works.
% Reframed to foreground the pipeline as the core contribution; distillation is a secondary study.
Dexterous multi-fingered hands are hardware-ready, yet there is no off-the-shelf recipe for training grasping policies on a new hand--arm system that transfer from simulation to real hardware without extensive retuning.
This thesis presents a dexterous grasping pipeline for a 23-degree-of-freedom Agile Hand on a Franka Research~3 arm.
A privileged teacher policy is trained with PPO under automatic domain randomization.
The teacher reaches 85.8\,\% grasp success at 12.1\,\% unsafe-episode rate.
The teacher is then distilled into an RGB-based student using SafeDAgger, which gates every action through a safety criterion and hands control to the teacher whenever the student deviates too far from the teacher policy.
SafeDAgger and DAgger reach comparable grasp success (72.7\,\% vs.\ 71.7\,\%), but SafeDAgger maintains a 50\,\% lower unsafe-episode rate during training.
Together, these results establish a dexterous grasping pipeline and indicate that safety-aware distillation matches task performance while substantially reducing training-time risk.%

% Zusammenfassung Deutsch:
% Struktur nach Chi-Review: Problem - Lösung - Ergebnis. Keine Verweise auf verwandte Arbeiten.

Mehrfingrige geschickte Roboterhände sind hardwareseitig einsatzbereit, doch es fehlt an einem standardisierten Rezept, um Greifstrategien auf einem neuen Hand-Arm-System zu trainieren, die ohne aufwändiges Nachjustieren vom Simulator auf reale Hardware übertragbar sind.
Diese Arbeit stellt eine Pipeline für geschicktes Greifen mit einer 23-Freiheitsgrad Agile Hand auf einem Franka Research~3 Arm vor.
Ein privilegierter Lehrer wird mit PPO unter automatischer Domain-Randomization trainiert.
Der Lehrer erreicht 85{,}8\,\% Greiferfolg bei 12{,}1\,\% unsicheren Episoden.
Der Lehrer wird anschließend mittels SafeDAgger in einen RGB-basierten Schüler destilliert, wobei jede Aktion durch ein Sicherheitskriterium gefiltert und die Kontrolle an den Lehrer übergeben wird, sobald der Schüler zu stark von der Lehrerstrategie abweicht.
SafeDAgger und DAgger erreichen vergleichbaren Greiferfolg (72{,}7\,\% gegenüber 71{,}7\,\%), doch SafeDAgger hält eine um 50\,\% niedrigere Rate unsicherer Episoden während des Trainings.
Zusammen etablieren diese Ergebnisse eine Pipeline für geschicktes Greifen und zeigen, dass sicherheitsbewusste Destillation die Aufgabenleistung erhält und gleichzeitig das Trainingsrisiko erheblich senkt.%
