\section{Network Architecture}%
\label{app:network_architecture}%

The teacher policy uses an \ac{LSTM}-based recurrent network implemented in rl\_games.
The actor consists of an MLP encoder followed by an LSTM layer with 1024 hidden units, followed by an output head mapping to the 23-dimensional action space.
The critic has an identical architecture but receives the larger critic observation (\cref{tab:ac_observations}).

The student uses a vision--proprioception fusion architecture:
\begin{enumerate}
    \item \textbf{Visual encoder:} A ResNet-18 backbone processes each stereo image independently, producing a 32-dimensional feature vector per image.
    \item \textbf{Cross-attention fusion:} A cross-attention transformer module fuses the two stereo features into a single 64-dimensional visual embedding.
    \item \textbf{Policy head:} The visual embedding is concatenated with the proprioceptive state vector and fed into an \ac{LSTM} with 512 hidden units, followed by a linear output layer producing the 23-dimensional action.
\end{enumerate}

\section{Environment Scaling and ADR-Randomised Distillation}%
\label{app:scaling_adr}%

Building on the primary distillation comparison, two follow-up experiments isolate the effect of environment count and physics randomisation during distillation.
Both reuse the same teacher, loss, and architecture; each modifies a single variable.

\paragraph{32 environments (4 per object).}
Increasing the parallel environment count from 24 to 32 yields the strongest student in this work: 79.2\,\% \acs{GSR} on the 8-object evaluation set, reaching 92.3\,\% of the teacher ceiling.
\acs{UER} is essentially unchanged (64.6\,\% vs.\ 62.7\,\%).
The improvement is consistent with the teacher training experience: cleaner per-object gradients dominate the trade-off against per-iteration wall-clock cost.

\paragraph{Distillation under \acs{ADR}~5.}
Enabling moderate physics randomisation (ADR level~5) during distillation degrades both methods, but the relative drop differs sharply: SafeDAgger falls from 79.2\,\% to 63.9\,\% \acs{GSR} ($-15.3$\,pp), while DAgger falls from 71.7\,\% to 48.8\,\% ($-22.9$\,pp).
The DAgger student's collision rate roughly doubles, whereas SafeDAgger's grows more modestly.
This asymmetric degradation provides direct evidence that safety-aware gating is less brittle to distribution shifts than uniform-coverage imitation.
